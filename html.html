Tag

Tags are the markers that build up HTML. The ones that are enclosed inside the <>.

They can be an opening tag, a closing tag, or a self-closing tag. Which ones are self-closing is up to experience and memorization.

For example:

  • <p> is an opening p tag.
  • </p> is a closing p tag.
  • <img> is a self-closing img tag.

Element

An element is the entire HTML object, if you will. An element is made of its opening and closing tags, as well as its content.

Here's an example of a paragraph element.


<p>I'm a paragraph element! Both tags and this text form the element!</p>

Elements can also be inside of other elements.

Attribute

Attributes are the properties elements can have.

These values are assigned during the opening tag, and dictate the functionality or behavior of the element.

Example of an image element with an "src" and an "alt" attribute:


<img src="kkcyber.png" alt="This is an example image!">

Block

A block is a display type that causes the element to always take the entire width of the page. All other elements before and after it will be above or below.

Some examples of block elements are p, h1~h6, ul, ol, div, table.

Inline

Inline elements are those that flow with the text, and are aligned to it.

Some examples of inline elements are img, a, span.

Like class, this is actually an attribute, not a tag.

The id attribute can be added to any element to give it a unique identifier. Since it's unique, only 1 element in the entire page can have an id with the same name.

On its own, this does nothing, but it makes it easier to target an element using CSS or Javascript.

Using a link, you can also scroll directly to an element in the same page by setting the "href" attribute to the name of the id with a hashtag.


<a href="#boo">Scroll down!</a>
<h4 id="boo">Boo!</a>

Like id, this is actually an attribute, not a tag.

Unlike id, different elements can share the same class name.

On its own, this does nothing, but it makes it easier to target an element using CSS or Javascript.

Since multiple elements can share the same class name, they can be targeted together to assign similar properties or behavior.

In the following example, we use CSS to color all the elements red. Please see the css guide for more info.


<p class="redExample">This will be painted red!</p>
<p>This text is normal, except the word <span class="redExample">red</span>!</p>
<style>
    .redExample {
        color: red;
    }
</style>

A div is simply a block. You can put things inside of it and it will behave like a block.

This is not very useful on its own, but can be extremely powerful when combined with CSS and Javascript, as you can group things together.

Each of these tutorials are contained inside a div, and displayed using Javascript when clicking the buttons.


<div>This is loose text inside a div.</div>
This is loose text outside the div. It takes a new line because the div is a block.

A span is an inline element without behavior.

This is not very useful on its own, but can be extremely powerful when combined with CSS and Javascript, as it simplifies applying style or behavior to anything inside it.


<span>Each</span> <span>word</span> <span>is</span> <span>in</span> <span>a</span> <span>span</span>.

Tables are formed with several tags, but we will stick to the most basic and important: table, tr, td.

Tables can be very unwieldy, so it's preferred not to resort to them unless you actually need a table for displaying data.

For page structure, it's recommended to stick to other block elements (like div) and CSS.

table

This is the outermost tag of a table element. It contains everything else and delimits where it starts and ends.


<table></table>

tr

This is used to start a new row inside the table, and stands for table row.

A table element may contain multiple tr elements inside.


<table>
    <tr></tr>
    <tr></tr>
    <tr></tr>
</table>

td

This is used to place a new cell inside the table, and stands for table data. These go inside the tr elements, and contain the actual data of the table.

By setting the "colspan" and "rowspan" attributes to a number, you can make a cell take up the spots of multiple cells.

Cells fill from left to right, in order of appearance.


<table>
    <tr>
        <td></td>
        <td>Ally</td>
        <td>Betray</td>
    </tr>
    <tr>
        <td>Ally</td>
        <td>+1/+1</td>
        <td>-2/+3</td>
    </tr>
    <tr>
        <td>Betray</td>
        <td>+3/-2</td>
        <td>+0/+0</td>
    </tr>
</table>
